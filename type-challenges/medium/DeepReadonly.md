Solution

```ts
type DeepReadonly<T extends object> = {
  readonly [KeyT in keyof T]: T[KeyT] extends Object
    ? T[KeyT] extends Function | Promise<infer PromiseReturnType>
      ? T[KeyT] 
      : DeepReadonly<T[KeyT]>
    : T[KeyT]
}
```
